## 安装 

1. 在本地创建一个 blog 文件夹，
2. 直接下载GitHub仓库源码，解压到本地的目录下，或者 `git clone` 仓库。
3. 执行 `npm install` 安装所需要的插件。
4. 安装完成后执行 `hexo cl && hexo g && hexo s` 即可运行。
